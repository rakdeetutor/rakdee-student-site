เฉลยลิมิตของฟังก์ชัน

บทที่ 2 แคลคูลัสเบื้องต้น · หัวข้อ 2.1–2.2

ตอนที่ 1 ลิมิตพื้นฐาน — แทนค่าตรง
1
$\displaystyle\lim_{x\to 3}\bigl(2x^2 - 5x + 7\bigr)$
ตอบ$10$

พหุนามต่อเนื่องทุกที่ → แทน $x = 3$ ได้ทันที

$= 2(3)^2 - 5(3) + 7 = 18 - 15 + 7 = 10$

2
$\displaystyle\lim_{x\to -2}\bigl(x^3 + 4x + 3\bigr)$
ตอบ$-13$

$= (-2)^3 + 4(-2) + 3 = -8 - 8 + 3 = -13$

3
$\displaystyle\lim_{x\to 4}\sqrt{x + 5}$
ตอบ$3$

ฟังก์ชันรากต่อเนื่องเมื่อใต้รากเป็นจำนวนบวก → แทน $x = 4$

$= \sqrt{4 + 5} = \sqrt{9} = 3$

4
$\displaystyle\lim_{x\to 1}\frac{x^2 + 2}{x + 3}$
ตอบ$\tfrac{3}{4}$

ตัวส่วน $x + 3$ ที่ $x = 1$ ได้ $4 \ne 0$ → แทนได้ทันที

$= \dfrac{1 + 2}{1 + 3} = \dfrac{3}{4}$

ตอนที่ 2 รูปแบบ $\tfrac{0}{0}$ — แยกตัวประกอบ
5
$\displaystyle\lim_{x\to 3}\frac{x^2 - 9}{x - 3}$
ตอบ$6$
เช็คแทน $x = 3$ → $\tfrac{0}{0}$ ต้องแยกตัวประกอบ
แยก$x^2 - 9 = (x-3)(x+3)$ → ตัด $(x-3)$ เหลือ $x + 3$
ตอบ$\lim_{x\to 3}(x+3) = 6$
6
$\displaystyle\lim_{x\to -1}\frac{x^2 + 3x + 2}{x + 1}$
ตอบ$1$
เช็คแทน $x = -1$ → $\tfrac{0}{0}$
แยก$x^2 + 3x + 2 = (x+1)(x+2)$ → ตัด $(x+1)$ เหลือ $x + 2$
ตอบ$\lim_{x\to -1}(x+2) = 1$
7
$\displaystyle\lim_{x\to 2}\frac{x^3 - 8}{x - 2}$
ตอบ$12$
เช็คแทน $x = 2$ → $\tfrac{0}{0}$
สูตร$a^3 - b^3 = (a-b)(a^2 + ab + b^2)$ → $x^3 - 8 = (x-2)(x^2 + 2x + 4)$
ตอบ$\lim_{x\to 2}(x^2 + 2x + 4) = 4 + 4 + 4 = 12$
8
$\displaystyle\lim_{x\to 4}\frac{x^2 - 7x + 12}{x - 4}$
ตอบ$1$
เช็คแทน $x = 4$ → $\tfrac{0}{0}$
แยก$x^2 - 7x + 12 = (x-3)(x-4)$ → ตัด $(x-4)$ เหลือ $x - 3$
ตอบ$\lim_{x\to 4}(x-3) = 1$
9
$\displaystyle\lim_{x\to 1}\frac{x^4 - 1}{x - 1}$
ตอบ$4$
เช็คแทน $x = 1$ → $\tfrac{0}{0}$
แยก$x^4 - 1 = (x-1)(x^3 + x^2 + x + 1)$
ตอบ$\lim_{x\to 1}(x^3 + x^2 + x + 1) = 1 + 1 + 1 + 1 = 4$
ตอนที่ 3 รูปแบบ $\tfrac{0}{0}$ — คูณคอนจูเกต
10
$\displaystyle\lim_{x\to 0}\frac{\sqrt{x+4} - 2}{x}$
ตอบ$\tfrac{1}{4}$
เช็คแทน $x = 0$ → $\tfrac{0}{0}$ มีราก → คูณคอนจูเกต $\sqrt{x+4}+2$
คูณ$\dfrac{(\sqrt{x+4}-2)(\sqrt{x+4}+2)}{x(\sqrt{x+4}+2)} = \dfrac{(x+4)-4}{x(\sqrt{x+4}+2)} = \dfrac{x}{x(\sqrt{x+4}+2)}$
ตัด$= \dfrac{1}{\sqrt{x+4}+2}$ → $\dfrac{1}{\sqrt{4}+2} = \dfrac{1}{4}$
11
$\displaystyle\lim_{x\to 5}\frac{\sqrt{x-1} - 2}{x - 5}$
ตอบ$\tfrac{1}{4}$
คูณคูณคอนจูเกต $\sqrt{x-1}+2$ ทั้งบนล่าง: $\dfrac{(x-1)-4}{(x-5)(\sqrt{x-1}+2)} = \dfrac{x-5}{(x-5)(\sqrt{x-1}+2)}$
ตัด$= \dfrac{1}{\sqrt{x-1}+2}$ → $\dfrac{1}{\sqrt{4}+2} = \dfrac{1}{4}$
12
$\displaystyle\lim_{x\to 3}\frac{x - 3}{\sqrt{x+1} - 2}$
ตอบ$4$
คูณคูณคอนจูเกต $\sqrt{x+1}+2$: $\dfrac{(x-3)(\sqrt{x+1}+2)}{(x+1)-4} = \dfrac{(x-3)(\sqrt{x+1}+2)}{x-3}$
ตัด$= \sqrt{x+1}+2$ → $\sqrt{4}+2 = 4$
ตอนที่ 4 ลิมิตที่อนันต์
13
$\displaystyle\lim_{x\to\infty}\frac{4x^2 + 3x - 1}{2x^2 - 7}$
ตอบ$2$

ดีกรีตัวเศษ = ดีกรีตัวส่วน $(= 2)$ → ลิมิต = อัตราส่วนสัมประสิทธิ์ตัวนำ $= \tfrac{4}{2} = 2$

14
$\displaystyle\lim_{x\to\infty}\frac{3x + 5}{x^3 + 2}$
ตอบ$0$

ดีกรีตัวเศษ $(=1) <$ ดีกรีตัวส่วน $(=3)$ → ลิมิต $= 0$

15
$\displaystyle\lim_{x\to\infty}\frac{6x^3 - x^2 + 5}{2x^3 + 7x}$
ตอบ$3$

ดีกรีเท่ากัน $(= 3)$ → $\tfrac{6}{2} = 3$

16
$\displaystyle\lim_{x\to\infty}\frac{(2x+1)(3x-2)}{x^2 + 4}$
ตอบ$6$
กระจาย$(2x+1)(3x-2) = 6x^2 - x - 2$
ตอบ$\dfrac{6x^2 - x - 2}{x^2 + 4}$ → ดีกรีเท่า → $\tfrac{6}{1} = 6$
ตอนที่ 5 ลิมิตซ้าย-ขวา (Piecewise)
17
$f(x) = \begin{cases} 2x+1, & x < 1 \\ x^2+2, & x \ge 1 \end{cases}$
ตอบ(ก) $3$  (ข) $3$  (ค) $3$
ซ้าย$x \to 1^-$ ใช้ $2x+1$ → $2(1)+1 = 3$
ขวา$x \to 1^+$ ใช้ $x^2+2$ → $1 + 2 = 3$
รวมซ้าย $=$ ขวา $= 3$ → $\lim_{x\to 1}f(x) = 3$ (มีค่า)
18
$f(x) = \begin{cases} x^2-4, & x < 2 \\ 3x-5, & x \ge 2 \end{cases}$   หา $\lim_{x\to 2}f(x)$
ตอบไม่มีค่า
ซ้าย$x \to 2^-$ ใช้ $x^2-4$ → $4 - 4 = 0$
ขวา$x \to 2^+$ ใช้ $3x-5$ → $6 - 5 = 1$
รวมซ้าย $\ne$ ขวา $(0 \ne 1)$ → ลิมิตไม่มีค่า (กราฟมีจุดกระโดด)
19
$f(x) = \dfrac{|x - 3|}{x - 3}$
ตอบ(ก) $-1$  (ข) $1$  (ค) ไม่มีค่า
ซ้าย$x < 3$ → $|x-3| = -(x-3)$ → $f(x) = \dfrac{-(x-3)}{x-3} = -1$
ขวา$x > 3$ → $|x-3| = x-3$ → $f(x) = 1$
รวมซ้าย $\ne$ ขวา → ลิมิตไม่มีค่า

เทคนิคจำ: $\dfrac{|x-a|}{x-a}$ ใกล้ $x = a$ จะเป็น $-1$ ฝั่งซ้ายและ $+1$ ฝั่งขวาเสมอ → ไม่มีลิมิต

ตอนที่ 6 ความต่อเนื่อง
20
$f(x) = \begin{cases} \tfrac{x^2-16}{x-4}, & x \ne 4 \\ k, & x = 4 \end{cases}$   ต่อเนื่องที่ $x = 4$
ตอบ$k = 8$
เงื่อนไขต่อเนื่องที่ $x = 4$ ต้องมี $\lim_{x\to 4}f(x) = f(4) = k$
แยก$\dfrac{x^2-16}{x-4} = \dfrac{(x-4)(x+4)}{x-4} = x+4$
ตอบ$\lim_{x\to 4}(x+4) = 8$ ∴ $k = 8$
21
$f(x) = \begin{cases} ax+b, & x \le 2 \\ x^2-1, & x > 2 \end{cases}$   ต่อเนื่องที่ $x = 2$ และ $f(1) = 4$
ตอบ$a = -1,\ b = 5$
ต่อเนื่องที่ $x = 2$: $a(2)+b = 2^2-1 \Rightarrow 2a+b = 3$  …(1)
f(1)$f(1) = a(1)+b = 4 \Rightarrow a + b = 4$  …(2)
แก้(1) − (2): $a = -1$   แทนใน (2): $b = 5$

ตรวจ: $2(-1) + 5 = 3$ ✓   $f(1) = -1 + 5 = 4$ ✓

22
$f(x) = \begin{cases} \tfrac{\sqrt{x+7}-3}{x-2}, & x \ne 2 \\ k, & x = 2 \end{cases}$   ต่อเนื่องที่ $x = 2$
ตอบ$k = \tfrac{1}{6}$
เช็คแทน $x = 2$: $\sqrt{9}-3 = 0$, $x-2 = 0$ → $\tfrac{0}{0}$
คูณคอนจูเกต $\sqrt{x+7}+3$: $\dfrac{(x+7)-9}{(x-2)(\sqrt{x+7}+3)} = \dfrac{x-2}{(x-2)(\sqrt{x+7}+3)}$
ตัด$= \dfrac{1}{\sqrt{x+7}+3}$ → $\dfrac{1}{\sqrt{9}+3} = \dfrac{1}{6}$
ตอบ∴ $k = \tfrac{1}{6}$
