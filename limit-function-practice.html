สูตรและเทคนิคสำคัญ
  • แทนค่าตรงได้ → ใช้ทันที (พหุนาม / ตรรกยะที่ตัวส่วน ≠ 0)
  • เจอ $\tfrac{0}{0}$ → แยกตัวประกอบ หรือ คูณคอนจูเกต
  • ลิมิตที่ $\infty$ → เทียบดีกรีพหุนาม / หารด้วยกำลังสูงสุด
  • ลิมิตซ้าย = ลิมิตขวา → ลิมิตมีค่า
  • ต่อเนื่องที่ $a$: ครบ 3 ข้อ → $f(a)$ มีค่า, $\lim$ มีค่า, $\lim = f(a)$
  • $|x-a|$ ที่ $x=a$ → ใช้ลิมิตซ้าย-ขวา (ค่า -, +)
