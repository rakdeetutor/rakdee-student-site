Section 1

ลำดับอนันต์คืออะไร

รู้จักนิยาม · เขียนสัญลักษณ์ · แยกประเภท

📐 นิยาม

ลำดับอนันต์ คือฟังก์ชันที่มีโดเมนเป็นเซตของจำนวนนับ $\mathbb{N} = \{1, 2, 3, \dots\}$

เขียนแทนด้วย $\{a_n\}$ หรือ $a_1, a_2, a_3, \dots$

โดย $a_n$ คือ พจน์ทั่วไป (general term) — สูตรของพจน์ที่ $n$

💡 Insight ของครูเต้ย

ลำดับ = ฟังก์ชัน (มอง $n$ เป็น input · $a_n$ เป็น output) — แต่ ขาดตอน เพราะ $n$ เป็นจำนวนนับเท่านั้น ไม่มี $a_{1.5}$

🏷 ประเภทที่ต้องรู้จัก

ประเภทสูตร $a_n$ตัวอย่าง
เลขคณิต$a_1 + (n-1)d$2, 5, 8, 11, ...
เรขาคณิต$a_1 \cdot r^{n-1}$3, 6, 12, 24, ...
ฮาร์มอนิก$1/n$1, 1/2, 1/3, ...
กำลังสอง$n^2$1, 4, 9, 16, ...
สลับเครื่องหมาย$(-1)^n$-1, 1, -1, 1, ...

เลือกสูตร $a_n$ แล้วดูจุดบนกราฟ

จำนวนพจน์: 20

Section 2

แนวคิดของลิมิต — ลู่เข้า vs ลู่ออก

เข้าใจว่า "เข้าใกล้" หมายความว่าอะไร

📐 นิยามแบบเข้าใจง่าย

กล่าวว่า ลำดับ $\{a_n\}$ ลู่เข้าหา $L$ ถ้าเมื่อ $n$ มีค่ามากขึ้นเรื่อยๆ พจน์ $a_n$ จะเข้าใกล้ค่า $L$ ได้ใกล้เท่าใดก็ได้

$\displaystyle \lim_{n \to \infty} a_n = L$

ถ้าหา $L$ ที่เป็นจำนวนจริงไม่ได้ — เรียกว่า ลู่ออก (divergent)

🏷 รูปแบบลู่ออก 3 แบบ

แบบพฤติกรรมตัวอย่าง
ออกสู่ $+\infty$โตไม่มีขีดจำกัด$a_n = n$
ออกสู่ $-\infty$ลดลงไม่มีขีดจำกัด$a_n = -n^2$
แกว่ง (oscillate)ไม่เข้าค่าใด$a_n = (-1)^n$
💡 ครูเต้ยพูดว่า

นักเรียนชอบสับสนระหว่าง "$a_n \to 0$" กับ "$a_n = 0$" — ลิมิตคือ ปลายทาง ไม่ใช่ค่าจริงของพจน์ใดพจน์หนึ่ง

เลื่อน $n$ เพื่อดูว่า $a_n$ เข้าใกล้เส้นแนวนอน $y = L$ แค่ไหน

n สูงสุด: 15

เฉลย: ข้อ 3 — เพราะ $(1/2)^n \to 0$ ดังนั้น $a_n \to 3$

Section 3

ลิมิตพื้นฐานที่ ต้องจำให้ได้

3 สูตรหลัก · ใช้ตอบโจทย์ 80% ของข้อสอบ

สูตร 1
$\displaystyle \lim_{n\to\infty} c = c$

ค่าคงตัวยังคงเป็นค่าคงตัวเสมอ

สูตร 2
$\displaystyle \lim_{n\to\infty} \frac{1}{n^k} = 0$ ($k > 0$)

หัวใจของโจทย์ตรรกยะทั้งหมด

สูตร 3 (สำคัญที่สุด)
$\displaystyle \lim_{n\to\infty} r^n = ?$

ขึ้นกับค่า $r$ — ดูตารางด้านล่าง

⚠️ ตาราง $r^n$ — ต้องท่องให้คล่อง

เงื่อนไข $r$$\lim r^n$ตัวอย่าง
$|r| < 1$0$(1/2)^n, (-0.3)^n$
$r = 1$1$1^n = 1$
$r > 1$หาค่าไม่ได้ ($+\infty$)$2^n, 1.01^n$
$r = -1$หาค่าไม่ได้ (แกว่ง)$(-1)^n$
$r < -1$หาค่าไม่ได้ (แกว่ง+โต)$(-2)^n$

กับดักคลาสสิค: นักเรียนเห็น $(-0.5)^n$ แล้วคิดว่าหาค่าไม่ได้เพราะติดลบ — ผิด! เพราะ $|-0.5| = 0.5 < 1$ ดังนั้นลิมิต $= 0$

Q3a. $\displaystyle \lim_{n\to\infty}\left(-\frac{2}{3}\right)^n = ?$

Section 4

ทฤษฎีบทลิมิต — Limit Laws

เครื่องมือ 5 ข้อ · ใช้แตกลิมิตเป็นชิ้นย่อย

ถ้า $\displaystyle\lim_{n\to\infty} a_n = A$ และ $\displaystyle\lim_{n\to\infty} b_n = B$ (ทั้งคู่ลู่เข้า) แล้ว:

1. Sum / Difference
$\lim (a_n \pm b_n) = A \pm B$
ตัวอย่าง: $\lim\left(\frac{1}{n} + 3\right) = 0 + 3 = 3$
2. Constant Multiple
$\lim (c \cdot a_n) = c \cdot A$
ตัวอย่าง: $\lim \dfrac{7}{n^2} = 7 \cdot 0 = 0$
3. Product
$\lim (a_n \cdot b_n) = A \cdot B$
ตัวอย่าง: $\lim \left(\frac{1}{n} \cdot \frac{1}{n}\right) = 0 \cdot 0 = 0$
4. Quotient (b ≠ 0)
$\lim \dfrac{a_n}{b_n} = \dfrac{A}{B}$ (ถ้า $B \neq 0$)
ใช้กับฟังก์ชันตรรกยะ — ดู Section 5
5. Power
$\lim (a_n)^k = A^k$ (เมื่อ $k$ เป็นจำนวนเต็มบวก)
ตัวอย่าง: $\lim \left(2 + \frac{1}{n}\right)^3 = 2^3 = 8$
⚠️ กับดัก!

ทฤษฎีบทใช้ได้ เฉพาะเมื่อทั้ง $a_n$ และ $b_n$ ลู่เข้า เท่านั้น

เช่น $\lim n - \lim n$ ไม่ใช่ $\infty - \infty = 0$ — รูปแบบนี้ หาค่าไม่ได้โดยตรง ต้องจัดรูปก่อน (Section 6)

📝 ตัวอย่างการใช้งาน

หาค่า $\displaystyle \lim_{n\to\infty}\left(5 - \frac{2}{n} + \frac{3}{n^2}\right)$

แตกลิมิตด้วยกฎ Sum: $= \lim 5 - \lim \dfrac{2}{n} + \lim \dfrac{3}{n^2}$
ใช้กฎ Constant Multiple: $= 5 - 2 \cdot \lim \dfrac{1}{n} + 3 \cdot \lim \dfrac{1}{n^2}$
ใช้สูตรพื้นฐาน $1/n^k \to 0$: $= 5 - 2(0) + 3(0)$
คำตอบ: $= 5$

Section 5

ลิมิตของฟังก์ชันตรรกยะ — เทคนิคหลัก

โจทย์ที่เจอบ่อยที่สุดในข้อสอบ A-Level

🔑 กฎ "หารด้วยกำลังสูงสุดของส่วน"

เมื่อพบ $\displaystyle\lim_{n\to\infty} \dfrac{p(n)}{q(n)}$ ที่เป็นพหุนาม — ให้หารทั้งเศษและส่วนด้วย $n^k$ โดย $k$ คือกำลังสูงสุดของ ส่วน

Case A: deg(เศษ) < deg(ส่วน)

ลิมิต $= 0$

$\lim \dfrac{n+1}{n^2+3} = 0$
Case B: deg(เศษ) = deg(ส่วน)

ลิมิต = อัตราส่วนสัมประสิทธิ์นำ

$\lim \dfrac{3n^2+1}{2n^2-n} = \dfrac{3}{2}$
Case C: deg(เศษ) > deg(ส่วน)

หาค่าไม่ได้ ($\pm\infty$)

$\lim \dfrac{n^3}{n+1}$ = หาค่าไม่ได้
⚡ Shortcut

เห็นโจทย์ตรรกยะ → เทียบดีกรีก่อน รู้คำตอบทันทีว่าจะเป็น 0, ค่าจริง, หรือหาค่าไม่ได้ — ค่อยทำงานละเอียดเฉพาะ Case B

Section 6

ลิมิตที่ ต้องจัดรูป ก่อน

รูปแบบ $\infty - \infty$ · เครื่องหมายติดราก · ขั้นสูง

🔑 รูปแบบ $\infty - \infty$ (ผลต่างที่ติดราก)

เช่น $\displaystyle\lim_{n\to\infty}\left(\sqrt{n^2+n} - n\right)$ — ใช้สูตร คอนจูเกต (conjugate) คูณทั้งบนและล่าง

หาค่า $\displaystyle \lim_{n\to\infty}\left(\sqrt{n^2+n} - n\right)$

คูณคอนจูเกต $\sqrt{n^2+n}+n$ ทั้งบนและล่าง: $$= \lim \frac{(\sqrt{n^2+n}-n)(\sqrt{n^2+n}+n)}{\sqrt{n^2+n}+n}$$
เศษเป็น $(a-b)(a+b) = a^2 - b^2$: $$= \lim \frac{(n^2+n) - n^2}{\sqrt{n^2+n}+n} = \lim \frac{n}{\sqrt{n^2+n}+n}$$
หารทุกพจน์ด้วย $n$: $$= \lim \frac{1}{\sqrt{1 + 1/n} + 1}$$
$1/n \to 0$ ดังนั้น: $$= \frac{1}{\sqrt{1+0}+1} = \frac{1}{2}$$
📌 จำได้สั้นๆ
  • เห็น $\sqrt{} - \sqrt{}$ หรือ $\sqrt{} - n$ → คูณคอนจูเกต
  • คอนจูเกตของ $a - b$ คือ $a + b$
  • หลังคูณแล้วใช้ $(a-b)(a+b)=a^2-b^2$ → รากหายไป
  • ค่อยหารด้วยกำลังสูงสุดของ $n$

🔑 รูปแบบ $\dfrac{r_1^n}{r_2^n}$ — หารด้วย $r$ ที่ใหญ่กว่า

หาค่า $\displaystyle \lim_{n\to\infty}\frac{2^n + 3^n}{3^n + 4^n}$

หารทั้งบน-ล่างด้วย $4^n$ (ใหญ่สุด): $$= \lim \frac{(1/2)^n + (3/4)^n}{(3/4)^n + 1}$$
$|r| < 1$ ทุกตัว → $\to 0$: $$= \frac{0+0}{0+1} = 0$$

✏️ Quick check

Q6. $\displaystyle \lim_{n\to\infty}\left(\sqrt{n^2+4n} - n\right)$ มีค่าเท่าใด?

คูณคอนจูเกต → $\dfrac{4n}{\sqrt{n^2+4n}+n}$ → หาร $n$ → $\dfrac{4}{\sqrt{1+4/n}+1} \to \dfrac{4}{2} = 2$

Section 7

อนุกรมอนันต์ — Infinite Series

การบวกพจน์ลำดับไม่รู้จบ — มี/ไม่มีคำตอบ

📐 นิยาม

กำหนดลำดับ $\{a_n\}$ — อนุกรมอนันต์ คือผลบวกทั้งหมด:

$\displaystyle \sum_{n=1}^{\infty} a_n = a_1 + a_2 + a_3 + \dots$

ผลบวกย่อย (partial sum): $S_n = a_1 + a_2 + \dots + a_n$

ผลบวกของอนุกรม:

$S = \displaystyle\lim_{n\to\infty} S_n$
  • ถ้า $\lim S_n$ มีค่าจริง → อนุกรม ลู่เข้า (convergent) มีผลบวก $S$
  • ถ้า $\lim S_n$ หาค่าไม่ได้ → อนุกรม ลู่ออก (divergent) ไม่มีผลบวก

⚠️ Test ลู่ออก (Divergence Test)

ถ้า $\displaystyle\lim_{n\to\infty} a_n \neq 0$ — อนุกรม $\sum a_n$ ลู่ออก เสมอ

ระวัง: ในทางกลับกัน ไม่ เป็นจริง — $\lim a_n = 0$ ไม่ได้ แปลว่าอนุกรมลู่เข้า (เช่น $\sum 1/n$ ลู่ออก แม้ $1/n \to 0$)

Section 8

อนุกรมเรขาคณิตอนันต์ — สูตรเด็ด

สูตรเดียวที่ ต้อง จำขึ้นใจ — ออกข้อสอบทุกปี

📐 สูตรหลัก

อนุกรมเรขาคณิต: $a + ar + ar^2 + ar^3 + \dots = \displaystyle\sum_{n=1}^{\infty} ar^{n-1}$

$\boxed{\,S = \dfrac{a}{1-r}\,}$ เมื่อ $|r| < 1$ เท่านั้น

เมื่อ $|r| \geq 1$ — อนุกรมลู่ออก ไม่มีผลบวก

📌 ใช้ยังไง 3 ขั้น
  1. หา $a$ (พจน์แรก)
  2. หา $r$ = พจน์ถัดไป $\div$ พจน์ปัจจุบัน
  3. เช็ค $|r| < 1$? — ถ้าใช่ ใช้สูตร · ถ้าไม่ ตอบ "หาผลบวกไม่ได้"

📝 ตัวอย่าง

หา $S$ ของ $4 + 2 + 1 + \dfrac{1}{2} + \dots$

$a = 4$ · $r = 2/4 = 1/2$
$|r| = 1/2 < 1$ ✓ ใช้สูตรได้
$S = \dfrac{4}{1 - 1/2} = \dfrac{4}{1/2} = 8$

🎯 Application: ทศนิยมซ้ำ

เปลี่ยน $0.\overline{27} = 0.272727\dots$ ให้เป็นเศษส่วน

เขียนเป็นอนุกรม: $\dfrac{27}{100} + \dfrac{27}{10000} + \dfrac{27}{1000000} + \dots$
$a = 27/100$ · $r = 1/100$
$S = \dfrac{27/100}{1 - 1/100} = \dfrac{27/100}{99/100} = \dfrac{27}{99} = \dfrac{3}{11}$

✏️ Quick check

Q8a. ผลบวก $9 + 3 + 1 + \dfrac{1}{3} + \dots = ?$

$a = 9, r = 1/3$ → $S = \dfrac{9}{1-1/3} = \dfrac{9}{2/3} = \dfrac{27}{2}$

Q8b. $0.\overline{6}$ เท่ากับเศษส่วนใด?

$a = 6/10, r = 1/10$ → $S = \dfrac{6/10}{9/10} = \dfrac{6}{9} = \dfrac{2}{3}$

สูตร · เทคนิค · กับดัก

หน้าเดียวจบ · กดพิมพ์เก็บไว้ทบทวน

📌 สูตรพื้นฐาน

  • $\lim c = c$
  • $\lim \dfrac{1}{n^k} = 0$ ($k > 0$)
  • $\lim r^n = 0$ ($|r| < 1$)
  • $\lim r^n = 1$ ($r = 1$)
  • $\lim r^n$ หาค่าไม่ได้ ($|r| > 1$ หรือ $r = -1$)

🧮 ทฤษฎีบทลิมิต (a, b ลู่เข้า)

  • $\lim(a_n \pm b_n) = A \pm B$
  • $\lim(c \cdot a_n) = c \cdot A$
  • $\lim(a_n \cdot b_n) = A \cdot B$
  • $\lim(a_n / b_n) = A/B$ ($B \neq 0$)
  • $\lim(a_n)^k = A^k$

⚡ ฟังก์ชันตรรกยะ — 3 Cases

  • เศษ < ส่วน: ลิมิต $= 0$
  • เศษ = ส่วน: สัมประสิทธิ์นำ ÷ สัมประสิทธิ์นำ
  • เศษ > ส่วน: หาค่าไม่ได้

วิธี: หารทุกพจน์ด้วย $n^k$ (k = ดีกรีสูงสุดของส่วน)

🔧 จัดรูปก่อนใส่ลิมิต

  • $\sqrt{} - \sqrt{}$ หรือ $\sqrt{} - n$: คูณคอนจูเกต
  • $r_1^n + r_2^n$ ในเศษและส่วน: หารด้วย $r$ ที่ใหญ่ที่สุด
  • $\infty - \infty$: รวม/แยกพจน์ก่อน

📐 อนุกรมเรขาคณิตอนันต์

$S = \dfrac{a}{1-r}$ เมื่อ $|r| < 1$
  • $a$ = พจน์แรก
  • $r$ = อัตราส่วนร่วม (พจน์ถัดไป ÷ พจน์ปัจจุบัน)
  • $|r| \geq 1$ → หาผลบวกไม่ได้

⚠️ กับดักที่ออกบ่อย

  • $(-0.5)^n \to 0$ (ติดลบไม่ใช่ปัญหา ขอแค่ $|r|<1$)
  • $\lim a_n = 0$ ไม่ได้ แปลว่า $\sum a_n$ ลู่เข้า
  • ทฤษฎีบทใช้ได้เฉพาะตอน ทั้งคู่ลู่เข้า
